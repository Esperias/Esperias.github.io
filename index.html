How to Play

Please select your platform to see the instructions!

  1. Press Win+R (Windows Key + R).
  2. Type in C:\Windows\System32\drivers\etc and press Enter.
  3. Right-click on the "hosts" file and open it with Notepad.
  4. Add the lines from the box on the right to the bottom of the file.
  5. Save the file. You may need administrator permissions.
  6. Open Growtopia and Connect!

We recommend using Hosts Go for the easiest setup.

  1. Download and Install HostsGo APK.
  2. Open Hosts Go and click on "Hosts Editor".
  3. Click the checkbox on the right, and click "Download Hosts File".
  4. Enter this URL: https://esperias.github.io/hosts.txt
  5. Click "Download and Apply".
  6. Return to the main screen and click "START".
  7. Open Growtopia and connect!

Alternative apps: Virtual Host, PowerTunnel.
